## Local Testing

To test the website locally:

1. Open `index.html` in a web browser, or
2. Use a local server:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx http-server
   ```
3. Navigate to `http://localhost:8000`
